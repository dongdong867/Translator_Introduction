Connecting Words
When translating a sentence or proverb, we should focus on how to connect each word instead of trying to translate every single word inside the proverb.


A Larger Database
It is possible for a translate system to correctively translate a proverb, but we need to create a huge database for the system to look into.


Story And Origin
Rather than directly translate the proverb, we should work on the task to make the translate system learn the story and the origin of the proverb.


Professional Models
We can create models for specific field such as Law, Finance, Enginerring and Medical. Training the system with artifical intellegence to make the system understand and view the pages like a human does.
